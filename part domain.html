HOSTING

Apa itu Hosting?

Hosting adalah layanan yang menyediakan ruang (storage) di internet untuk menyimpan semua file dan data website. Tanpa hosting, sebuah website tidak dapat diakses oleh siapa pun. Ibarat rumah, hosting adalah “tanah” tempat bangunan website berdiri.

Hosting biasanya berupa server, yaitu komputer besar yang beroperasi 24 jam nonstop agar website selalu online. File yang disimpan di hosting meliputi:

  • halaman HTML
  • gambar dan video
  • file CSS dan JavaScript
  • database
  • email server

Jenis Hosting

Shared Hosting, Cloud Hosting, VPS
